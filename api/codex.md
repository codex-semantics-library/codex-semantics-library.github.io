# Codex Library API

{% include shields.html package=site.data.packages.codex %}

This library is Codex's core. It regroups generic components for program static analysis.
These components can then easily be instantiated to run an analysis on most languages.
For instance, we have instantiated them for both C and binary code analysis.

For the full API documentation, see [Codex v1.0](/api/codex/v1.0/)
